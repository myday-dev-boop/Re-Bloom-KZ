# 🌷 Telegram-бот: доска объявлений цветов

Продавец заполняет анкету → ты модерируешь → объявление публикуется в канал.

---

## Как это работает

1. Продавец пишет боту `/start` → «Подать объявление»
2. Бот по шагам собирает: фото → название → цена → город → описание → телефон
3. Заявка приходит **тебе** (админу) с кнопками ✅ Одобрить / ❌ Отклонить
4. После одобрения объявление публикуется в **канал** с кнопками связи (WhatsApp / Telegram)
5. Продавец получает уведомление, что объявление одобрено

---

## Что нужно для запуска (3 вещи)

### 1. Токен бота
У тебя уже есть (от @BotFather).

### 2. Твой Telegram ID (ADMIN_ID)
Это нужно, чтобы заявки приходили именно тебе.
- Запусти бота → напиши ему `/id`
- Бот пришлёт твой ID (число) → это `ADMIN_ID`

### 3. Канал для публикации (CHANNEL_ID)
1. Создай канал в Telegram (Новый канал)
2. Сделай его **публичным** и задай имя (напр. `flowers_almaty`)
3. Добавь своего бота в канал как **администратора** (Управление каналом → Администраторы → Добавить → найди своего бота → дай право «Публикация сообщений»)
4. `CHANNEL_ID` = `@flowers_almaty` (с собачкой)

> Если канал приватный — CHANNEL_ID будет вида `-100xxxxxxxxxx`. Узнать можно, переслав любое сообщение из канала боту @userinfobot

## Запуск на Railway

### Шаг 1. Загрузи на GitHub
Создай репозиторий → загрузи все файлы:
```
bot.py
requirements.txt
runtime.txt
Procfile
nixpacks.toml
```

### Шаг 2. Railway
1. railway.com → New Project → Deploy from GitHub repo → выбери репозиторий
2. Вкладка **Variables** → добавь:
   ```
   TELEGRAM_TOKEN  = твой токен
   ADMIN_ID        = твой Telegram ID (из команды /id)
   CHANNEL_ID      = @твой_канал
   MISE_PYTHON_GITHUB_ATTESTATIONS = false
   ```
3. Railway соберёт и запустит бота

> Переменная `MISE_PYTHON_GITHUB_ATTESTATIONS=false` нужна, чтобы починить установку Python (известная проблема Railway).

## Порядок настройки (важно!)

1. Сначала запусти бота **только с токеном** (ADMIN_ID и CHANNEL_ID можно поставить временно)
2. Напиши боту `/id` → узнай свой ID
3. Создай канал, добавь бота админом
4. Впиши настоящие ADMIN_ID и CHANNEL_ID в Variables
5. Railway перезапустит — всё готово

---

## Команды бота

- `/start` — главное меню, подать объявление
- `/help` — как пользоваться
- `/id` — узнать свой Telegram ID (для настройки)
- `/cancel` — отменить заполнение анкеты

## Возможные проблемы

**Заявки не приходят на модерацию?**
- Проверь ADMIN_ID (должен быть твой ID числом)
- Напиши боту `/start` хотя бы раз (бот не может писать первым)

**Не публикуется в канал?**
- Бот должен быть **администратором** канала с правом публикации
- Проверь CHANNEL_ID (@имя для публичного, -100... для приватного)

**Ошибка Python при сборке?**
- Добавь переменную `MISE_PYTHON_GITHUB_ATTESTATIONS=false`
